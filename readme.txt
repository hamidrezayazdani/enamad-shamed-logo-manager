This plugin helps you to easily put the logo of E-namad, Shamed and Zarrinpal on your website

== Description ==

This plugin uses shortcodes and widgets to help you put the Enamad, Shamed and Zarrinpal logos in any part of your website that you like.

== Frequently Asked Questions ==

= Can I use this plugin with any WordPress theme?  =

Yes

= Where can I change the plugin settings?  =

Go to تنظیمات > تنظیمات لوگوی اینماد

= What is the shortcode of Enamad logo?  =

[enamadlogo_shortcode]

= What is the shortcode of Shamed logo?  =

[shamedlogo_shortcode]

= What is the shortcode of Zarrinpal logo?  =

[zarrinpallogo_shortcode]

= What is the shortcode of all of logos?  =

[ywp_esl_logos]
