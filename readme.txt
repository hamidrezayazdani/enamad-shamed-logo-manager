This plugin helps you to easily put the logo of E-namad and Shamed on your website

== Description ==

This plugin uses shortcodes and widgets to help you put the Enamad and Shamed logos in any part of your website that you like.

== Frequently Asked Questions ==

= Can I use this plugin with any WordPress theme?  =

Yes

= What is the shortcode of Enamad logo?  =

[enamadlogo_shortcode]YOUR ENAMAD CODE[/enamadlogo_shortcode]

= What is the shortcode of Shamed logo?  =

[shamedlogo_shortcode]YOUR SHAMED CODE[/shamedlogo_shortcode]
